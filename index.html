About Me

Hi, I'm Susmita. I'm a Frontend developer based in Hyderabad, India who loves creating great looking websites. Aside from web development, I enjoy listening to music, cooking and stalking beautiful looking websites.

Research current website design trends, marketing techniques and analyzing its impact on the audience. Create a prototype using pen and paper accordingly then move to design software to design it properly.

Once design reaches a certain aesthetics I code the project using my favorite text editor.

4. Optimize

In the end, I work on responsiveness and make website loading blazingly fast.
